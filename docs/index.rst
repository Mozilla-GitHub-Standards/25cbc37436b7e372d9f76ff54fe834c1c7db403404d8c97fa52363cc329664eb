Overview
========

Plugincheck is the software that runs a plugin directory with a
JSON API. It serves as the backend for the `Check Your Plugins page
<https://www.mozilla.org/plugincheck/>`_.

This is at heart a `Django <http://djangoproject.com>`_-powered
site. It uses a `PostgreSQL <http://www.postgresql.org/>`_ database
for data storage.
